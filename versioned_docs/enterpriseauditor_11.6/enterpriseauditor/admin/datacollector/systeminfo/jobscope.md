# SystemInfo: Job Scope

The Job Scope page is where to select whether or not scoping should be used during execution. It is a wizard page for the category of File Shares.

![System Info Data Collector Wizard Job Scope page](/img/versioned_docs/enterpriseauditor_11.6/enterpriseauditor/admin/datacollector/systeminfo/jobscope.png)

Select from the following options:

- Don’t Use Scoping
- Use Scoping
