Specify Sql User Name Window

The Specify Sql User name window is opened from a field where a SQL Server account is needed.

Enter the SQLÂ Server user name into the text box. Multiple user names can be added using a semicolon (;), a comma (,), or a space. Then click OK. The Specify Sql User name window closes, and the account is added to the field where the window was opened.
